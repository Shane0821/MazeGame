## 基于元气骑士风格的迷宫游戏

### 开发环境
VS2019 + Cocos2dx

### 游戏特点
- **完美迷宫**：我们使用广度优先搜索算法生成迷宫，确保从起点到终点有且仅有一条路径，无死胡同，每个位置都可达。
- **多关卡设计**：游戏分为多个关卡，难度逐步递增，为玩家提供不断挑战的机会。
- **时间限制**：每个关卡的时间限制基于起点到终点的路径设定，考验玩家的速度和策略。
- **失败提示**：如果玩家在规定时间内未能通关，游戏会显示正确的路径，帮助玩家理解并改进策略。
- **地图元素随机**：游戏的地图元素是随机的，这增加了游戏的丰富性。
